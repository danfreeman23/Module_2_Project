# Анализ лояльности пользователей Яндекс Афиши

## Описание проекта

Комплексный анализ поведения пользователей платформы Яндекс Афиша с целью выявления факторов, влияющих на повторные покупки и лояльность клиентов.

## Цели проекта

- Выявить характеристики пользователей, склонных к повторным покупкам
- Определить оптимальные стратегии удержания клиентов
- Разработать рекомендации для отдела маркетинга

## Ключевые результаты

### Основные инсайты:

1. **Оптимальный интервал** между покупками: 7-14 дней (75.7% лояльных пользователей)
2. **Лучшие типы мероприятий** для первого заказа: спортивные события и концерты
3. **Идеальный средний чек**: 400-750 RUB
4. **Оптимальное количество билетов**: 2-3 за заказ

### Статистика данных:
- Проанализировано: 21,143 пользователя
- Период данных: июнь-октябрь 2024
- Доля возвращающихся пользователей: 61.9%
- Доля лояльных пользователей: 28.6%

## Структура проекта

- `notebooks/` - Jupyter Notebook с полным анализом
- `src/` - вспомогательные Python-модули
- `data/` - данные проекта (не включены в репозиторий)
- `reports/` - финальный отчет и презентации
